
# Beyond Product

An Ecommerce for the people


## Tech Stack

**Client:** React, Redux, Bootstrap

**Server:** Node, Express

  
## Installation

Install Beyond Product with npm

```bash
// Backend
cd Backend
npm install or npm i
npm start
```


```bash
//FrontEnd
cd FrontEndd
npm install or npm i
npm start
```
